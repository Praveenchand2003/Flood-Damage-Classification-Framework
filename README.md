# Flood-Damage-Classification-Framework

## Dataset

Dataset Link:

https://www.kaggle.com/datasets/dhawalsrivastava2583/flood-classification-dataset

## Requirements

Install the following libraries:

```bash
pip install tensorflow numpy pandas matplotlib seaborn scikit-learn
```

## How to Run

### Step 1

Download the dataset from Kaggle.

### Step 2

Open the notebook:

Flood_Damage_Classification_Framework.ipynb

### Step 3

Add the Flood Classification Dataset to the Kaggle Notebook.

### Step 4

Enable GPU:

Settings → Accelerator → GPU

### Step 5

Run all notebook cells from top to bottom.

### Step 6

The notebook will:

* Load the dataset
* Preprocess images
* Apply data augmentation
* Train a Custom CNN model
* Train a MobileNetV2 Transfer Learning model
* Generate accuracy and loss curves
* Generate confusion matrices
* Generate classification reports
* Evaluate model performance

### Step 7

Review the final results and generated figures.
